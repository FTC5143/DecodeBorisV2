TROUBLESHOOTING QUICK REFERENCE
================================

If compilation fails:
→ Check import statements
→ Verify class names match file names
→ Check syntax errors
→ Run "Clean Project" then "Rebuild Project"

If hardware not detected:
→ Check device names in config exactly match code
→ Verify I2C/USB connections
→ Check device powered on
→ Verify device drivers installed

If ball detection fails:
→ Check HuskyLens firmware updated
→ Verify detection algorithm selected
→ Check camera pointed at balls
→ Verify adequate lighting

If autonomous doesn't move:
→ Check ball detection working
→ Press Y button to enable auto mode
→ Check telemetry for current state
→ Verify motors have power

If turret won't aim:
→ Check manual turret control works
→ Check encoder connection
→ Verify turretPIDCoef tuned
→ Check power supply voltage

If no haptic feedback:
→ Check enableHapticFeedback = true
→ Test gamepad vibration separately
→ Check gamepad firmware

If performance lag:
→ Reduce telemetry output
→ Check USB/I2C bus congestion
→ Verify battery voltage
→ Profile code for hot spots

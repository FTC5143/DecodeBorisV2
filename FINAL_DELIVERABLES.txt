COMPONENT CLASSES (4 files - 1,990 lines of code)
─────────────────────────────────────────────────

1. BallDetectionVision.java (470 lines)
   Location: xcentrics/components/live/
   Purpose: Husky Lens ball detection integration
   Features:
   • Real-time ball detection and tracking
   • Multi-ball prioritization
   • Distance-based filtering
   • Connection status monitoring
   • Time-stamped detection data
   
   Public API:
   - getClosestBall() → nearest ball
   - getAllDetectedBalls() → sorted list
   - getBallCount() → detection count
   - areBallsDetected() → boolean check
   - isConnected() → connection status

2. AprilTagVision.java (490 lines)
   Location: xcentrics/components/live/
   Purpose: AprilTag detection with pose fusion
   Features:
   • Real-time pose estimation
   • Vision + odometry fusion (70/30 weighted)
   • Pose history smoothing
   • Dual alliance support (Red/Blue)
   • Validity checking and filtering
   
   Public API:
   - getPose() → fused pose estimate
   - getAveragePose() → smoothed pose
   - isPoseValid() → freshness check
   - isRedTagVisible() / isBlueTagVisible()
   - setStreaming(boolean) → performance control

3. BallPickupAndScoringSystem.java (570 lines)
   Location: xcentrics/components/live/
   Purpose: Main autonomous state machine
   Features:
   • 10-state finite state machine
   • Automatic timeout protection
   • Path generation and following
   • Turret readiness checking
   • Multi-shot firing sequence
   • Full error recovery
   
   Public API:
   - setAutoEnabled(boolean) → toggle mode
   - isAutoEnabled() → mode check
   - getCurrentState() → current FSM state
   - update(LinearOpMode) → main loop

4. AutonomousController.java (460 lines)
   Location: xcentrics/components/live/
   Purpose: Driver input and haptic feedback
   Features:
   • Button input handling (Y, X, Back)
   • Haptic feedback patterns
   • Emergency stop execution
   • Mode override capability
   • Rich status telemetry
   
   Public API:
   - setAutoEnabled(boolean) / isAutoModeActive()
   - triggerHapticFeedback(HapticPattern)
   - sendReadyToFireNotification()
   - forceDisableAuto()
   - getCurrentAutoState()

🎮 DRIVER CONTROLS
──────────────────
Y Button: Toggle autonomous mode ON/OFF (haptic feedback)
X Button: Override/cancel autonomous (emergency)
Back Button: EMERGENCY STOP (all systems halt)

🤖 AUTONOMOUS STATES
──────────────────
MANUAL → SCAN → NAVIGATE_TO_BALL → PICKUP → NAVIGATE_TO_GOAL → 
ALIGN_TURRET → WAIT_FOR_READY → SHOOT → VERIFY → (loop)

📊 CONFIGURATION
────────────────
Ball Detection: minBlockSize, maxBallDistance
Vision Fusion: fusionWeight, maxPoseUpdateDistance
State Machine: Timeouts for each state
Turret Control: Position ±50 ticks, Velocity ±100 RPM
